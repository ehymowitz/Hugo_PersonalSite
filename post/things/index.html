Graphics

I like using Java to make cool visualizations! Here are few of the ones I’ve made more recently.

This one uses the haplet. Each ball has its own force which keeps it attached to the nearby balls. When you move haplet through the balls, you not only feel the weight of each ball - you also get an idea of the force keeping all the balls together. I thought it looked pretty cool!

This is a pretty basic visualization, just a bunch of balls of rings expanding and contracting with different frequencies.

After making this visualization, I integrated it into the haplet to make it interactive. Moving the Haplet around changes parameters of the rings.
